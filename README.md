# InsightCAP Community

InsightCAP Community is the public community repository for InsightCAP.

This repository contains public documentation, release information, and selected community-facing materials. Some InsightCAP components are proprietary and are not included here.

## Status

This repository is currently prepared as a private staging repository before public release.

## Repository Contents

- Public product overview
- Community Edition installation notes
- High-level architecture documentation
- Privacy and local-data notes
- Licensing information
