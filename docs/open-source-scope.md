# Open Source Scope

This repository is the public community repository for InsightCAP.

## Included

- Public documentation
- Community Edition license information
- High-level architecture notes
- Installation notes
- Selected community-facing source code, if added later

## Not Included

- Private source code
- Proprietary modules
- Commercial Edition features
- Hosted services
- Internal prompts, policies, and implementation details
- Private roadmap and planning documents
- Personal data, local caches, databases, and generated artifacts
