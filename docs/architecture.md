# InsightCAP Architecture

This document provides a high-level public overview of InsightCAP Community.

InsightCAP is designed as a local-first desktop application. The Community Edition focuses on user-controlled capture, organization, and retrieval workflows.

## High-Level Components

- Desktop shell: provides the local desktop application experience.
- User interface: presents capture, review, search, and knowledge workflows.
- Local services: coordinate local data processing and application state.
- Storage layer: keeps user data on the local device where applicable.
- Release package: distributes signed or packaged Community Edition builds through GitHub Releases.

## Local-First Direction

The Community Edition is intended to keep personal knowledge workflows understandable and user-controlled. Public documentation describes the architecture at a high level and does not include proprietary implementation details.

## Proprietary Components

Some InsightCAP components are not included in this repository, including commercial features, private source code, proprietary modules, internal policies, and non-public implementation details.
