# Privacy and Data

InsightCAP Community is documented as a local-first desktop application.

## Public Privacy Notes

- This repository does not include private user data.
- Local cache files, databases, environment files, and generated artifacts should not be committed.
- Public releases should not include personal device names, local usernames, private emails, local paths, tokens, API keys, or machine-specific metadata.

## Pre-Release Review

Before public release, repository contents and release artifacts should be reviewed for:

- Personal email addresses
- Local user paths
- Device names
- Operating system build details
- Tokens and secrets
- Databases and cache files
- Internal-only documentation
