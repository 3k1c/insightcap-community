# Installation

InsightCAP Community Edition installers will be distributed through GitHub Releases.

## Download

Use the latest release from:

https://github.com/3k1c/insightcap-community/releases

## Notes

- Installers and binary artifacts should be downloaded from Releases, not committed directly to the repository.
- Verify the release notes before installing a new version.
- Community Edition binaries are free to use for personal and commercial purposes under the Community Edition License.
