be uploaded as a GitHub Release asset and should not be committed directly to

## Build Refresh

This installer refresh includes startup and recovery stability fixes:

- Defers local embedding initialization so the first screen appears sooner.
- Opens the migration/recovery flow instead of exiting when the active
  knowledge base cannot be opened during startup.
- Verifies and repairs recovered knowledge-base encryption state before saving
  the restored login credentials.
